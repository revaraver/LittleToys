下面四个分别是导出计划, 其中q是只导出能显示的,qh是导出所有的包括隐藏的  
	**/q (Query)  /qh (Query Hidden)**
powercfg /q 381b4222-f694-41f0-9685-ff5bb260df2e > C:\balance.txt
powercfg /q 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c > C:\high.txt
powercfg /qh 381b4222-f694-41f0-9685-ff5bb260df2e > C:\balance_all.txt
powercfg /qh 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c > C:\high_all.txt

比较重要的几个反隐藏
1.**解锁并调整 EPP 参数（解除保守的升频限制）** (Energy Performance Preference)
powercfg -attributes SUB_PROCESSOR PERFEPP -ATTRIB_HIDE

2.**处理器性能提升阈值**
powercfg -attributes SUB_PROCESSOR PERFINCTHRESHOLD -ATTRIB_HIDE

3.**处理器性能增强策略（决定使用睿频的积极性）**  
powercfg -attributes SUB_PROCESSOR PERFBOOSTPOL -ATTRIB_HIDE

4.**处理器性能时间检查间隔（决定系统评估负载的反应速度，单位为毫秒）**  
powercfg -attributes SUB_PROCESSOR PERFCHECK -ATTRIB_HIDE

# <font color=#b84747>恢复相关的命令:</font>
下面的命令没有经过测试, 不太建议使用
**1. 一键解锁所有隐藏电源选项（全部可见）：**  
注：请右键点击开始菜单，选择“Windows PowerShell (管理员)”或“终端 (管理员)”运行。

codePowershell

```
Get-ChildItem -Path "HKLM:\System\CurrentControlSet\Control\Power\PowerSettings" -Recurse | ForEach-Object { $attr = Get-ItemProperty -Path $_.PSPath -Name "Attributes" -ErrorAction SilentlyContinue; if ($attr -ne $null -and $attr.Attributes -eq 2) { Set-ItemProperty -Path $_.PSPath -Name "Attributes" -Value 0 } }
```

(原理解释：去注册表电源目录遍历，把所有属性值为 2 (代表隐藏) 的项，全部强制改成 0 (代表显示)。执行后，你的高级电源设置里会瞬间多出六七十个选项！)

**2. 一键恢复所有隐藏选项（恢复原厂整洁）：**

codePowershell

```
Get-ChildItem -Path "HKLM:\System\CurrentControlSet\Control\Power\PowerSettings" -Recurse | ForEach-Object { $attr = Get-ItemProperty -Path $_.PSPath -Name "Attributes" -ErrorAction SilentlyContinue; if ($attr -ne $null -and $attr.Attributes -eq 0) { Set-ItemProperty -Path $_.PSPath -Name "Attributes" -Value 2 } }
```
